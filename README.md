# Excel Data Analytics Portfolio 

This repository contains my final project and practice sheets tracking my complete data analyst preparation course.
 What's Inside These Files->
* **Complete Chapter Roadmap:** Practical implementation of all chapters from scratch, starting from structural data cleaning, advanced logic (`IFS`, `AND/OR`), dynamic text operations, and custom timeline calculus (`NETWORKDAYS.INTL`).
* **Advanced Multi-Criteria Analytics:** Deep-dive queries utilizing core operational formulas like `SUMIFS`, `AVERAGEIFS`, and `MAXIFS` to extract transactional peaks across products and regions.
* **Dynamic Sales Dashboard:** Built an interactive executive dashboard layout directly connected to dynamic charts, timelines, automated macro triggers (`SlicerConnection`), and interactive slicers.
* **The Dataset:** All calculations, models, and dashboard visualizations flow dynamically from the raw transactional data stored in the sheet named **`raw data`**.
